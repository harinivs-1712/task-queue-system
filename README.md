Task Queue System

A scalable backend task queue system built using Node.js, Express, and MongoDB.
The system supports priority-based job scheduling, delayed task execution, retries, concurrency control, stuck job recovery, and graceful shutdown handling.

🚀 Features
Background job processing
Priority-based task scheduling
Delayed/scheduled tasks using runAt
Retry mechanism for failed jobs
Concurrency control
Atomic job claiming using MongoDB
Recovery of stuck jobs
Graceful shutdown handling
REST API support
Docker support
🛠 Tech Stack
Node.js
Express.js
MongoDB
Mongoose
Docker
